# 简介

Runixo 是一个 AI-Native 服务器管理平台，采用 **客户端 + 服务器 Agent** 架构。

```
┌──────────────────────────────────────────┐
│            Runixo Client (Electron)      │
│  Vue 3 + Element Plus │ AI 模块 │ 管理   │
└──────────────┬───────────────────────────┘
               │ gRPC (TLS 加密)
┌──────────────▼───────────────────────────┐
│            Runixo Agent (Go 单二进制)     │
│  gRPC 服务 │ 命令执行器 │ 数据采集器      │
└──────────────────────────────────────────┘
```

## 核心特性

- **安全架构** — Agent 不暴露 Web 端口，所有通信 TLS 加密
- **AI 深度融合** — 自然语言运维、智能诊断、自动化工作流
- **多服务器管理** — 多节点、批量操作、跨节点编排
- **容器管理** — Docker 容器/镜像/Compose 全生命周期管理
- **插件生态** — 插件市场，功能即装即用
- **实时监控** — CPU、内存、磁盘、网络实时指标

## 环境要求

- **客户端**：Windows / macOS / Linux
- **Agent**：Linux（推荐）、macOS、FreeBSD
- **开发**：Node.js 18+、Go 1.22+、pnpm 8+
